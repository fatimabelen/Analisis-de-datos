Manejo de valores faltantes¶

El manejo de los valores faltantes puede implicar eliminarlos o reemplazarlos por otro valor, y se hace segn la columna, es decir, no se pueden tratar todos los valores faltantes del dataframe de la misma forma. En este caso los valores faltantes de se reemplazan por la media en ambas columnas.

La ventaja de reemplazar por la media es que el valor de la media en la columna no se ve afectado por los valores faltantes.

Para reemplazar los valores faltantes por la media se utiliza el mtodo fillna(), con el parmetro value para indicar el valor que reemplaza a los valores faltantes, y el parmetro inplace=True para indicar que el reemplazo se haga en la columna asignada.

Al igual que en el mtodo describe(), en el mtodo fillna() se calcula el promedio de cada columna sobre los valores existentes sin tener en cuenta los valores faltantes.

Tener en cuenta que se debe aplicar el mtodo a cada columna para poder reealizar el reemplazo de los valores faltantes con el mtodo fillna(), si aplicaramos el mtodo al dataframe tendriamos como resultado el reemplazo de TODOS los valores faltantes de dataframe por el mismo promedio.

In [22]:
df1['Aos de experiencia'].fillna(value=df1['Aos de experiencia'].mean(), inplace=True)
df1['Sueldo'].fillna(value=df1['Sueldo'].mean(), inplace=True)

Diagrama de dispersión - Análisis de los datos de ambas columnas¶

Los diagramas de dispersin muestran de manera grfica la relacin que hay entre dos variables cuantitativas. En este caso, se utiliza para observar si hay una relacin lineal entre los aos de experiencia y el sueldo percibido.

Para realizar el grfico de dispersin se utiliza el mtodo scatter() de Matplotlib, y para graficar la lnea de relacin se utiliza el mtodo polyfit() de Numpy.

In [62]:
x = df1["Aos de experiencia"] # Eje x
y = df1["Sueldo"] # Eje y


pend, inter = np.polyfit(x, y, 1) #Ajusto la lnea de relacin

plt.scatter(x, y) # Diagrama de dispersin
plt.plot(x, pend*x + inter, color='r') # Grafico la lnea de relacin en el diagrama
plt.xlabel("Aos de experiencia")
plt.ylabel("Salario")
plt.show()

A simple vista se puede observar que hay una relacin lineal directa entre las dos variables. Esto se puede verificar calculando la covarianza y el coeficiente de correlacin.

La covarianza es una medida descriptiva que permite determinar el tipo de relacin lineal entre dos variables. Y se calcula utilizando la funcin cov() de Numpy.

El coeficiente de correlacin se utiliza para medir la magnitud de la relacin lineal entre dos variables, es decir, indica cun fuerte o dbil es una relacin lineal. Y se calcula con la funcin coef_corr() deNumpy.

In [64]:
covarianza = np.cov(x,y) 
cov = round(covarianza[0,1], 2) 
print('La covarianza es: ' + str(cov))

La covarianza es: 101993.47

El valor positivo de la covarianza indica que la relacin lineal entre los aos de experiencia y el sueldo es directa, es decir, cuando aumentan los aos de experiencia tambin lo hacen los sueldos.

In [65]:
coeficiente_corr= np.corrcoef(x,y)

print('El coeficiente de correlacin es: ' + str(coef_corr))

El coeficiente de correlacin es: 0.98

El valor del coeficiente de correlacin cercano a 1 indica una correlacin fuerte, es decir, que la relacin lineal es directa y fuerte.
